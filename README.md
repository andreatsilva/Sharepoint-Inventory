Sharepoint-Inventory
Shell script that makes an inventory of every file in every website and file in sharepoint.

Important Notice:

This script is APP Based, so an app needs to be created in MSEntra before running the script.

1.How to run:

- .\Inventory-Sharepoint.ps1 `
  -TenantId "" `
  -AppId "" `
  -CertificateThumbprint ""
